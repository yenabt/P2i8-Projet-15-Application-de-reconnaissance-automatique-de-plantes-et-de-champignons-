# P2i8-Projet-15-Application-de-reconnaissance-automatique-de-plantes-et-de-champignons-
L'objectif principal de ce projet
est de concevoir et de développer une application
mobile conviviale qui permettra aux utilisateurs de
prendre une photo d'une plante ou d'un champignon,
puis de recevoir instantanément des informations
précises sur l'espèce identifiée. Les objectifs
spécifiques incluent :
* Concevoir une interface utilisateur intuitive pour
l'application mobile, permettant aux utilisateurs de
capturer facilement des images et de recevoir des
résultats d'identification.
* Développer un système de traitement d'image robuste capable d'extraire des descripteurs
significatifs tels que la couleur, la texture et la forme à partir des images capturées.
* Mettre en œuvre un algorithme de comparaison efficace pour faire correspondre les descripteurs
extraits avec une base de données préexistante d'espèces végétales et fongiques.
* Intégrer des fonctionnalités supplémentaires telles que la recherche par nom d'espèce, la
classification par famille botanique ou fongique, et la fourniture d'informations complémentaires
sur les plantes et les champignons identifiés.
